Listas en HTML

Las listas son útiles para organizar información de manera clara y estructurada.

Lista Desordenada (ul)

Una lista desordenada se utiliza para agrupar elementos sin un orden específico.

  • Elemento de lista 1
  • Elemento de lista 2
  • Elemento de lista 3

Lista Ordenada (ol)

Una lista ordenada se utiliza para agrupar elementos en un orden específico.

  1. Elemento de lista ordenada 1
  2. Elemento de lista ordenada 2
  3. Elemento de lista ordenada 3

Ejemplos

Lista de la compra

  • Leche
  • Pan
  • Huevos

Orden de aprendisaje desarrollo web

  1. HTML
  2. CSS
  3. JavaScript
  4. Frameworks (React, Angular, Vue)

Lista de definiciones

HTML
Lenguaje de marcado para estructurar contenido en la web.
CSS
Lenguaje de estilo para diseñar y dar formato a páginas web.
JavaScript
Lenguaje de programación para crear interactividad en la web.
